# wallet-connectivity
The scripts collected in this repository were produced during various testing cycles involving wallet connectivity, transaction preparation, and contract interaction. Many of the files started as temporary utilities and later remained useful for debugging and verification purposes.

Certain examples investigate transaction environments that resemble perpetual markets. These experiments are intended to observe execution behavior, timing differences, and operational states rather than reproduce complete trading systems.

Another area of interest is maintaining interactions in a non-custodial manner. Several utilities avoid external dependencies and keep transaction handling localized in order to simplify testing and reduce operational complexity.

Some examples also examine how a token holder may interact with contract functions under different conditions. These experiments focus on execution flow, transaction records, and repeatable testing scenarios.

Because the repository evolved gradually, there is no single architecture shared by every file. Each utility reflects the problem it was originally written to solve, which results in different implementation styles throughout the project.
